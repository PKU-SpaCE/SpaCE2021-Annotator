# SpaCE2021-Annotator

SpaCE2021 标注工具

此仓库存放 SpaCE2021 标注工具 的前端代码。

**【注意】** 当前位于 main 分支，代码较旧，**dev分支才是最新代码**！
（划分不同分支而未合并的原因： dev 分支具有审核功能， main 分支不具备，而不同用户需要访问的页面种类不一样，所以当时没有把 dev 分支合并到 main 分支。）

技术说明：

- 代码采用 Vue 直接在 html 中编写，没有使用打包工具。
- 此标注工具没有后端。
- 下载代码后，用浏览器打开 index.html 后即可使用。
